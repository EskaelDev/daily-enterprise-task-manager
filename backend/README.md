## Installation
`> npm i` install all node packages.

`> npm run dev` for nodemon (service that listens for updates in files).

`> npm run build` to build .js files.

`> npm start` to start nerver from .js files.

## Info
API port: 5000

## Available endpoint
GET: localhost:5000/users
